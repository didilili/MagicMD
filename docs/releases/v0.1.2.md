# MagicMD v0.1.2

v0.1.2 是一个很小的发布版本，重点是把 MagicMD 的 Agent Skill 正式放到可安装目录里。

## 这次升级了什么

- 新增 `skills/magicmd/SKILL.md`，作为正式可安装的 MagicMD Skill。
- 新增 `skills/magicmd/agents/openai.yaml`，补充 Skill 在 Agent UI 里的显示信息。
- 保留根目录 `SKILL.md` 作为兼容入口，并指向正式 Skill 目录。
- README 和开发文档改为使用 `skills/magicmd` 作为安装路径。
- 新增测试，确保 Skill 发布目录不会被误删。

## 安装 Skill

从 GitHub 安装时使用：

```text
Repository: didilili/MagicMD
Skill path: skills/magicmd
```

如果使用 Codex 的 skill installer，可以运行：

```bash
python3 ~/.codex/skills/.system/skill-installer/scripts/install-skill-from-github.py \
  --repo didilili/MagicMD \
  --path skills/magicmd
```

安装后重启 Codex，让新 Skill 生效。

## 包版本

这个版本不改变文章抓取和 Markdown 转换核心行为。PyPI / npm 版本同步到 `0.1.2`，方便 CLI、npm wrapper、GitHub tag 和 Skill 发布状态保持一致。

# MagicMD v0.1.2

v0.1.2 is a small release that makes the MagicMD Agent Skill available from a clean installable directory.

## What's Changed

- Added `skills/magicmd/SKILL.md` as the publishable MagicMD Skill.
- Added `skills/magicmd/agents/openai.yaml` for Agent UI metadata.
- Kept the root `SKILL.md` as a compatibility entrypoint pointing to the publishable Skill.
- Updated README and maintainer docs to use `skills/magicmd` as the install path.
- Added a metadata test so the publishable Skill directory is not accidentally removed.

## Install the Skill

Use this GitHub source:

```text
Repository: didilili/MagicMD
Skill path: skills/magicmd
```

With Codex's skill installer:

```bash
python3 ~/.codex/skills/.system/skill-installer/scripts/install-skill-from-github.py \
  --repo didilili/MagicMD \
  --path skills/magicmd
```

Restart Codex after installing the Skill.
