# 配置生成器

选择发布目标、终端语言、文件命名和媒体路径，右侧会实时生成 `.magicmd.toml`。你可以快速定制内容包目录、Markdown 文件名、metadata、报告、CLI 输出语言和媒体路径；生成结果只使用 MagicMD 已经支持的字段，可以复制到项目根目录直接使用。
