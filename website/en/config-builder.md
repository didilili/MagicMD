# Config Builder

Choose a publishing target, terminal language, filenames, media paths, and optional GitHub publishing settings. Quickly customize package folders, Markdown filenames, metadata, reports, CLI output language, media links, and the content repository settings reused by `magicmd publish github`. The generated `.magicmd.toml` only uses fields already supported by MagicMD, so you can copy it to your project root and run it directly.
